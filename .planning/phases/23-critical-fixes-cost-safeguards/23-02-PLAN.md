This is a human-only action. Claude cannot configure Supabase billing — the dashboard has no public API for spend cap and alert configuration.

Configure two billing safeguards in the Supabase dashboard for the draftpokemon.com project:

Step 1 — Enable Spend Cap:
1. Open https://supabase.com/dashboard
2. Select the draftpokemon.com project
3. Go to Settings -> Billing
4. Find "Spend Cap" and enable it
5. Set the cap value. Recommended: $50/month for beta (adjust based on your Pro plan allowance)
6. Save the cap

Step 2 — Configure Billing Alert:
1. In the same Billing section, find "Alerts" or "Budget Alerts"
2. Create a new alert
3. Set the alert threshold BELOW the spend cap (e.g., if cap is $50, alert at $25 or $30)
4. Set the notification email to your address
5. Save the alert

Step 3 — Confirm both are saved and note the values for the SUMMARY.
